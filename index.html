  1. Go to scoop.sh
  2. Install scoop
  3. Once installed, install ffmpeg using scoop
  4. Then download Jugux
  5. Finished
[ Actually, you can use anything to install ffmpeg, as long as it can be used in the terminal, because it's required to run programs that involve videos. ]
